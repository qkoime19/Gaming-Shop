# Gaming Shop

Mini project for a game item shop built with HTML, CSS, and JavaScript.

## Features

- Product data stored in JavaScript arrays and objects
- Dynamic product cards
- Coin balance
- Inventory list
- Product images
- Rarity system
- Loot box
- Card animations
- Player avatar
- Search, category filtering, and sorting
- Limited items and discounts
- Stackable inventory quantities

## Files

- `index.html`
- `style.css`
- `script.js`
